* 简介

本仓库用来存放S1部分高楼的纯文本备份，以50楼为一份文件，请善用搜索功能。

本仓库仅存放7日内有回复的3页及以上帖子。若3天内没有回复，该帖子的纯文本备份会被挪到2023至今历史存档中。
