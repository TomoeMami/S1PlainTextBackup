一、自用安全配置

1.家庭局域网用syncthing手动同步资料，不外网传输，数据完全私有化。

2.微信通过局域网定期备份到电脑，手机丢失后可大部分恢复。

3.锁屏密码 + PIN 码双重锁

4.全平台关闭小额免密，所有支付、购物 APP 强制密码 / 指纹验证。

5.熟记电信 / 联通 24 小时官方紧急挂失流程：

电信卡：借任意手机拨打 10000 → 转人工 → 报手机号 + 姓名 + 身份证号 → 申请紧急挂失停机，立即生效

联通卡：借任意手机拨打 10010 → 转人工 → 报手机号 + 姓名 + 身份证号 → 申请紧急挂失停机，立即生效

二、手机丢失应急全流程

1.先冻结手机号：按上面流程，拨打对应运营商客服，双卡同时紧急挂失停机，原 SIM 卡立即失效，彻底切断验证码。

2.同步冻结账号：在微信 / 支付宝安全中心，输入丢失手机号 + 本人实名信息，无需登录、无需原手机验证码，远程一键冻结。

3.携带身份证办理新 SIM 卡，解冻账号、修改密码，恢复局域网三端同步，从电脑端还原微信聊天记录。

自己现有配置，冻结手机和微信部分是AI推演的，我也妹有验证。这是模拟半夜手机发现被盗了，得第二天早上去营业厅办理。
