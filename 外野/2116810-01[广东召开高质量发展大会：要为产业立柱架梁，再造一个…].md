####  马自达明天倒闭  
##### 1#       楼主       发表于 2023-1-28 20:46

“扎扎实实抓好今年，抓好5年，再深耕10年、30年，必定能再造一个新广东、再创让世界刮目相看的新奇迹。”1月28日是兔年首个工作日，上午9时，广东省委、省政府召开新春开年第一会——全省高质量发展大会，广东省委书记黄坤明在会上如是说。
澎湃新闻（[www.thepaper.cn](http://www.thepaper.cn)）从大会现场获悉，此次大会主会场参会人数达1000人，其中参会企业超500家。同时，大会以视频直播形式开至全省各地市、县（市、区），分会场参会人数达2.5万人，是近年来广东省召开的规模最大的会议。
大会上，广东多位省直部门“一把手”、各地市市委书记、企业代表等纷纷发言表态，展望新一年高质量发展的目标和计划。
<img src="https://pics3.baidu.com/feed/6159252dd42a283418b9dc66fd586be114cebfbd.jpeg@f_auto?token=e175b1708a346cf2c942f77e662b5142" id="aimg_S0S8K" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

广东高质量发展大会现场。 羊城晚报 图黄坤明表示，农历新春上班开工的第一天召开全省高质量发展大会，就是要在春意盎然的日子擂起奋进催征的金鼓，奏响走在前列、当好示范的强音。他还提出，在危与机、稳与进、攻与守的纵横捭阖中，全省上下形成了强烈共识，广东经济的“危”源自高质量发展不足，“机”要靠高质量发展才能紧紧抓住，贯彻新发展理念、推动高质量发展是广东的根本出路，“要把握高质量发展根本要求，着力推动质量变革、效率变革、动力变革。”
展现广东的拳拳用心和有力行动
党的**报告提出，“高质量发展是全面建设社会主义现代化国家的首要任务。”
2023年是贯彻落实党的**精神的开局之年，1月28日，广东全省高质量发展大会正式开幕。《南方日报》在当天头版刊发题为《岭南处处是春天——写在全省高质量发展大会召开之际》的文章。文中指出，春节后上班第一天，全省高质量发展大会就在广州召开，足见高质量发展在广东工作全局中的重要位置，足见省委、省政府抢抓时间、抢抓机遇，积极动员全省上下快速投入高质量发展的拳拳用心和有力行动。
澎湃新闻获悉，此次大会围绕“重点项目重大平台”“制造业当家”“百县千镇万村高质量发展工程”“五外联动”等主题深入探讨，共谋发展思路和举措。
<img src="https://pics0.baidu.com/feed/d62a6059252dd42af76504b2a2d6f9bec8eab8b0.jpeg@f_auto?token=8e9073e5c0e40debd9be08e9629a6abc" id="aimg_V3OwK" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

广州塔是广东发展的标志性成果之一。 视觉中国 图广东省发改委主任、省大湾区办主任艾学峰在大会上表示，要加强经济运行监测调度和稳增长政策统筹，协调有关部门落实社消零、进出口、投资等支撑指标，确保实现广东省2023年地区生产总值增长5%以上。艾学峰还表示，要推动重要投资项目的落地落实，发挥有效投资在高质量发展中的关键作用。要加快实施“十四五”规划确定的重大项目，确保全年固定资产投资增长8%、基础设施投资和工业投资实现两位数以上增长，省重点项目完成投资1万亿元。具体项目方面，艾学峰表示，今年要建成投用广汕汕高铁、珠三角水资源配置工程等一批重大项目，加快广湛、梅龙高铁，惠州机场，陆丰核电等项目建设，力争开工建设漳汕、梅武高铁，珠三角枢纽机场等项目，全年新增铁路建设里程共866公里等。
加快加力推动现代产业体系立柱架梁
以制造业为主体的实体经济是广东的立省之本、强省之基。步入新的一年，广东如何做优做强这份厚实“家当”？
在全省高质量发展大会上，广东省工业和信息化厅党组书记、厅长涂高坤说，目前，广东省制造业总产值预计突破16万亿元大关，拥有70余万户制造业企业法人和一大批优质企业、8个万亿级战略性产业集群。并且已经构建了制造当家的大产业、大平台、大项目、大企业、大环境，以及指挥体系、目标体系、政策体系、工作体系、考核体系的5+5的战略要求和工作体系机制。
涂高坤表示，接下来，广东将加快加力推动现代产业体系的立柱架梁，着力推动20个战略产业集群的建设，提质壮大现有的8个万亿级产业集群，发力行业新增3个万亿元级、4个以上的五千亿元级集群。争取2023年全省工业投资要增长达到10%以上，发挥工业投资对工业经济发展的放大叠加和倍增的作用。
<img src="https://pics0.baidu.com/feed/50da81cb39dbb6fddba7dcaaadc90913952b37b2.jpeg@f_auto?token=bb694715f0d263b3f6b9bddca11b1488" id="aimg_IZxH4" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

广东将加快加力推动现代产业体系的立柱架梁。 视觉中国 图此外，涂高坤还介绍，广东将努力营造大环境、贴身服务企业，把工信部门打造成为制造业企业的“娘家”，推动科技、人才、资金、土地、能源等资源要素向制造业集中配置，持续构建亲清政商关系，坚定不移致力于营造市场化法治化国际化一流营商环境，让企业敢干、敢闯、敢投。农业农村的发展是全省高质量发展的坚实支撑。广东省农业农村厅党组书记刘棕会则表示，全省农业农村系统将坚决贯彻省委、省政府工作部署，聚焦实施“百县千镇万村高质量发展工程”，抓重点补短板强弱项，以时不我待、只争朝夕的责任感、紧迫感全面推进乡村振兴、加快建设农业强省、一体推进农业农村现代化，为全省高质量发展释放巨大创造动能、消费潜能和发展空间，把全省高质量发展的“压舱石”压得更稳、更实。
大部分发言时间留给“一线”干部和企业代表
澎湃新闻注意到，1月28日，南方都市报发表《把舞台留给一线，给高质量发展更大空间》的评论，其中提到一个细节：在上午的主会场议程中，省领导的讲话时间并不太长，大部分的发言时间留给了各地市、关键厅以及各领域代表性企业。特别是会议将近一个小时的时间分配给了各种形式的企业经营者代表发言。可以说，此次会议把舞台留给了一线。
广东省委书记黄坤明在讲话中说：“今天安排各地市、各单位和各方代表作表态发言，就是要比决心、比干劲，比勇挑大梁的胆识和能力，展现开局的势头、新春的气象，在全省掀起齐心协力抓发展的热潮。”
在地市谈高质量发展发言环节，广东省副省长、珠海市委书记吕玉印表示，珠海将坚定不移把高质量发展作为首要任务，把制造业当家作为头号工程，坚定不移大抓产业第一，壮大实体经济；聚焦“4+3”产业体系，全面打响“产业项目落地攻坚年”行动，持续打好产业立柱、招商引资、园区提升、土地整备、强核赋能、政府流程再造六大攻坚战，“加快建设产业大市、产业强市，高质量建设新时代中国特色社会主义现代化国际化经济特区，努力成为全省发展又一重要引擎和珠江口西岸核心城市。”
广东省政协副**、佛山市委书记郑轲说：“走老路，去不到新的目的地。新春上班第一天，广东省委、省政府就展现新气象，没有互相走访、没有作揖拜年，而是吹响高质量发展的新号角。”
郑轲介绍，2022年经济工作挑战多，今年也不会少，但今年肩负挑大梁的更高目标，必须拼出佛山功夫，不走老路，勇攀新路，确保GDP增长6%、工业投资增长15%、进出口增长5.5%，把“高质量发展”、“制造业当家”干成稳中求进、走在前列的升腾景象。
他还提到，今年，佛山生产了一款石湾公仔叫“奔跑兔”，未来佛山将坚持以“奔跑兔”比速度，用快干、实干来改作风、树新风。本次大会之后，佛山将派出招商、招工、暖企、抢订单四支队伍奔赴全球各地。
作为广东第四个GDP破万亿的城市，2023年，东莞宣布将坚持制造业当家，坚持“科技创新+先进制造”，争取实现全年增长6%的目标。广东省人大常委会副主任、东莞市委书记肖亚非在会上表示，东莞把2023年定为“招商年”，实施“全员招商”，力争东莞的战略性新兴产业、振兴基地产值破千亿元。肖亚非还透露，东莞将投入2.4亿元支持企业春节前后稳工促产，全面提振民营企业信心。
值得一提的是，此次大会的主会场参会人数达1000人，其中参会企业超500家。多位与会企业代表表示，对广东高质量发展充满信心，将继续投资广东、扎根广东。
<img src="https://pics5.baidu.com/feed/3ac79f3df8dcd100a0fa9e65c666e51bb8122f0c.jpeg@f_auto?token=222b9f0f96c4e0236999d373660691bc" id="aimg_cFz87" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

广东高质量发展大会现场。图片来源：羊城晚报作为首位发言的企业经营者代表，中国电子信息产业集团有限公司董事长曾毅在会上表示，中国电子将在投资、建设粤港澳大湾区中，展现中央企业的责任和担当，全年投资计划增长30%，利润增长25%，充分发挥身处大湾区的战略优势，加速打造国家网信事业的核心战略科技力量。“入驻粤港澳大湾区一年多来，我们深深体会到湾区处处是春天。在新的一年里，中国电子将以这次会议为重大战略契机，更加创新、更加务实，深化战略合作，加快建设世界级湾区，发展最好的湾区，为全力推动高质量发展贡献中国电子的力量。”曾毅说。
而以家电制造业起家的美的集团股份有限公司董事长兼总裁方洪波上台发言时表示，身处机遇与挑战并存的时代，美的对广东坚持制造业专家充满信心，将坚定不移地推动制造业高质量发展，勇敢地走在向前增长的道路上，努力在下一轮博弈和竞争中建立新优势。
方洪波还表示，今年，美的将加快美的总部三期、数字科技产业园、制冷科技创新园、机电工业4.0基地、库卡科技园、南海小塘基地、顺德储能产业基地等先进制造和新产业项目的建设和投产，助力广东产业体系升级发展。
