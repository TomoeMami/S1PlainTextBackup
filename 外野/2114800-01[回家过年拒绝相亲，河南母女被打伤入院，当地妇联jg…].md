####  马自达明天倒闭  
##### 1#       楼主       发表于 2023-1-13 20:18

返乡过年拒绝相亲，河南34岁女子和母亲被打伤入院，当地妇联和jg已介入调查。

1月12日晚，上游新闻（报料邮箱：baoliaosy@163.com）记者收到网络爆料，称河南省驻马店市确山县双河镇一34岁女子，因为返乡过年拒绝相亲，被对方一家上门辱骂，并受到殴打，自己和母亲还被打伤住院。13日上午，受伤女子告诉上游新闻记者，这件事发展成现在这样自己也是“有点接受无能”。据悉，目前jg已经开始调查，确山县妇联告诉记者，当天上午镇妇联已经介入此事。

<img src="https://pics4.baidu.com/feed/a2cc7cd98d1001e963423fd2ccaff4e755e79719.jpeg@f_auto?token=43c79b8d37e21d81108be23f351e14d1" id="aimg_M6DmP" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

▲王女士脸部和颈部多处受伤。

“派出所民警上午就来了”，13日中午，上游新闻记者联系上了刚刚在医院做完笔录的王红丽，她透露，自己在广东一家民营企业从事会计工作，跟打人者董某某之前曾是一个村的，互相都认识，自己还和对方女儿是发小，平时并没有什么矛盾。

王红丽说，此前董某某就来家里，想为自己亲戚的儿子介绍对象，被自己明确拒绝。7日，董某某又来家里，与王红丽母亲闫有珍发生了争吵，“当时我正在外地，后来知道董某某来我家说了一些难听的话，说我30多岁还没成家，给我介绍的人也在外面打工也很能挣钱等等，而且话已经说出去了，又被我拒绝，让他在亲戚面前很没有没面子。”

<img src="https://pics1.baidu.com/feed/d1160924ab18972b0eb23085946cf4829f510a25.jpeg@f_auto?token=e4b0e1bfbf7402b6b804e58df99acf62" id="aimg_pmU6W" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

▲王女士脸部和颈部多处受伤。

“我已经明确拒绝了介绍，然后他还这样到家里当着父母的面来说我，我觉得很没有道理。”11日，王红丽就到董某某家想沟通一下，“董某某也没有接话，但让我没想到的是，他老婆和女儿的道歉方式竟然是当场给我下跪，吓了我一跳，我立刻把她们拉起来了，因为觉得这不太合适，就离开了。”

11日下午，董某某夫妇带着儿子和女儿，连续两次上门，“沟通过程也不太愉快，各说各的理，但董某某儿子董某觉得他家人下跪道歉是我逼的，就指着鼻子骂我，骂得很难听。”

“我怎么也没想到12日早上会发生那种事。”当天早上，董某某夫妇又来到王红丽家，双方再度发生争吵，“我被吵醒之后，也出来了，后来越吵越激动的董某某还一把掐住了我的脖子，我爸妈赶紧上来拉。”“可能是感觉自己只有两人有些吃亏，董某某又叫来了儿子董某，没想到董某提着一根钢管就来了，把我妈踢倒在地上，还把我妈打伤了，董某还跺了我一脚。”

<img src="https://pics6.baidu.com/feed/4034970a304e251f0b81de5bdc27461c7e3e534f.jpeg@f_auto?token=f5ff81c5f36df82a3e7ce325ccaae3ce" id="aimg_UOCt0" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

▲董某持铁棍殴打王女士母亲。

王红丽表示，她在事发当时已报警，自己和母亲随后都住进了信阳信钢医院接受治疗，在王红丽展示的伤势照片中，显示自己脸部、嘴唇、脖子等多处被抓破，自己还有些头晕，在医院输液。

王红丽母亲闫有珍的伤势较为严重。上游新闻记者看到了医院急诊科出具的CT诊断报告，影像意见为：左侧多发肋骨骨折，左侧少量胸腔积液，腰椎T12椎体前缘骨折。

<img src="https://pics7.baidu.com/feed/2cf5e0fe9925bc31678d681e2a7e02bac9137094.jpeg@f_auto?token=de99106df3b7be02078f033765adeb91" id="aimg_tb9pq" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

▲被打伤的王女士母亲。

上游新闻记者随后拨打了当地派出所的电话，但一直处于通话中的状态。随后记者联系上了确山县妇联，一位工作人员表示当天上午就已经收到了相关信息，双河镇妇联工作人员已经介入，因为事情才发生，仍需要时间才能了解具体情况。

上游新闻记者也试图联系董家核实相关情况，但一直未能打通王红丽发小董女士的电话。

<img src="https://pics2.baidu.com/feed/faf2b2119313b07edb2d2d0272761e2896dd8c50.jpeg@f_auto?token=1d1579d177605f1158cc31871b3b5fd1" id="aimg_l878h" onclick="zoom(this, this.src, 0, 0, 0)" width="656"/)

▲被打伤的王女士母亲的诊断报告。

“上午董家夫妻又追来医院，和我们吵了一架，还打扰到了其他病友”，王红丽无奈地告诉记者，“我真的没有想到会因为（拒绝相亲）这么一件事，就把人打伤住医院了，我现在甚至都有点接受无能。”
