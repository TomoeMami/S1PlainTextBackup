去年10月，知名洞潜区、搅动泥沙区、临时起意区、传来噩耗区UP主“神秘园”，发布了他的第一期鳌太线户外事故视频。在那期视频中，“神秘园”以原本能够获救，却最终遗憾离世的53岁吴先生为主线，呈现了鳌太线恶劣的天气和复杂的救援情况。
随后，和吴先生同期前往鳌太线登山的陈某放，成为“神秘园”鳌太线第二期的主角，被困8天后，陈某放凭借吃烂野果、树叶、过期腐乳奇迹生还。
几个月后，“神秘园”又制作了一期鳌太线上因天气突变，造成违法穿越者死伤失踪事故的视频节目。这三期关于“鳌太线”的视频，仅在B站就获得了累计1000万以上的播放量，也让UP主“神秘园”的知名度迅速提升。这些源于真实故事的解说视频，展现惊心动魄的遇险故事同时，也呈现着自然的残酷和人性的抉择。
看“神秘园”的视频，让网友在全知视角建构下，领略了荒野奇观的同时，洞悉了生存和救助的策略。更重要的是，看完视频，人们似乎重新夺回了对生活的掌控力。网友纷纷评论称：“看了半年神秘园，我觉得自己强得可怕。”“神秘园告诉了我们，半山腰没有卖烤肠的山不要爬。”
“非法穿越鳌太线，活下来罚款5000元，死了能上神秘园。”1月初，“神秘园”接受中国新闻周刊专访时表示：“不出意外，自己再也不会去操作关于鳌太线的选题了。”<strong>以下来自“神秘园”的口述：</strong><strong>别去“鳌太线”</strong>很多网友不知道，同一个地点的选题，我不会制作超过三次，罗布泊也好，鳌太线也好，我应该不会再做了，尤其是鳌太线。
其实，鳌太线我只做过三期视频，但给网友留下了很深的印象，这样就出现了一个问题，网上只要出现鳌太线的内容，大家就会下意识地认为是我做的。这一方面增加了我的知名度，一方面也增加了鳌太线的知名度。
但这让我感到警觉，并进行反思，决定不再做鳌太线了。
不再做的原因首先是从选题容量上来说，三期已经足够从各种维度把一个区域的复杂情况说清楚了。
其次，也是很重要的一点，我是真怕你们去。<img src="https://p.sda1.dev/30/27a23bfafb812cdbb5a69f82b76cfd90/640.png" referrerpolicy="no-referrer">">
我知道，有些人看完我的视频后觉得自己“强得可怕”，甚至对鳌太线“了如指掌”。
你们甚至已经学会了，要从塘口村潜入，到达2800营地后，从北坡能快速下撤到白云峡，从南坡下撤到核桃坪。
但是下雨不能走，道路会被山洪冲垮，最终还要返回2800营地。
要想活命需要前往大爷海找救援，但大爷海往返要四天，2800营地等救援不能乱跑，否则会与救援人员擦肩而过。
保温毯要贴身穿，羽绒服要买上万的，里外穿三层。
要带卫星通信电话，食物要带够10天的，4块压缩饼干不能丢，火种、哨子都要带。
遇险不能下撤，一定原地等待救援，同时收集虫子充饥。
最后，下山后去派出所自首缴纳5000元罚款。
但我想说，这些远远不足以支撑鳌太线的复杂和凶险，鳌太线真正离谱的地方是它分分钟就会变化出离谱的样貌，风、雨、雪、温度……所有的变化都是出其不意的，是骗、是偷袭、是不讲武德的。
鳌太线一年四季，每个季节各有各的“死法”。
我认为，网友看了我做了这么多期视频，应该早已经打消了这种冒险作死的冲动才对。
就像有的网友说的那样，看完“神秘园”，从卧室走到厕所，都要担心“失温”，赶紧把秋裤套上。这样做就很好，这是一个成年人应有的思考方式。
看几个视频就去拿生命冒险，这事过于离谱了。我的本意是希望大家能通过看我的视频对户外和自然多一些敬畏之心，而不是成为我视频中的主角。
你们要相信我，这绝非我的本意。<img src="https://p.sda1.dev/30/cf79ab57597edd4f24d1273bb482f758/640" id="aimg_pJaAH" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

一些网友看完视频总结出来的“速通鳌太线”的指南我也看到了，直接去派出所表示自己非法穿越完了，缴纳5000元罚款，随后拍照留念，以示“穿越成功”。
不得不说，从安全的角度这样做是可行的，但是您有这5000块钱，能买500多瓶没过期的腐乳。
最近我注意到了一条留言是这么写的：“当年徒步轻装速穿鳌太线，那年冬天，大雪封山，我一个人，没有团队，没有高级装备，没有报备，没有告诉家人，也没去。”
看完我笑了，只能说这种好人，一生平安是有原因的。
有人问我，为什么不下场和这些热梗互动，我第一是真看不过来，第二是我觉得大家能自己玩起来是件好事，我下场性质就变了。
<strong>一条被反复走错的路</strong><strong>
</strong>去年，茶冷公路和火星公路走错的内容我破天荒地做了4期。
那是两条位于青海无人区雅丹地貌中的两条路，其实都是废弃公路，但是由于景色特殊，这些年吸引了很多自驾游的人们前往。
绝大部分人的目的地都是火星公路，而迷路的原因则是在工商银行石油小镇遗址岔口误入了茶冷公路。而在戈壁无人区中陷车后，高温、缺水、没有通信信号，都会造成非常危险的情况，这也给当地公安机关救援制造了很大的困难。
发完第一期视频后，没过多久又发现有人在那里遇险，甚至社交媒体上很多人发布自己曾经走错路的消息，这让我对这个选题再次提起了兴趣。
为了制作相关的视频，我反复对比了当地的情况、道路的细节，和具体案例走岔路的原因，终于发现和确定了其中的规律。<img src="https://p.sda1.dev/30/a61f57d0f20c0be33660554006e23d85/640" id="aimg_f43Ts" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)“神秘园”视频中强调易走错的位置 图/视频截图
自驾游客从冷湖镇出发，沿国道215行驶7公里到达火星一号公路起点，再行驶7公里到达工商银行石油小镇遗址，在这里分成两条路，往右继续是火星一号公路，往左则是茶冷公路。
两条路的宽度、路面看上去基本一样，就连此后的公路里程碑数也都是7、8、9、10等，只是茶冷公路的里程碑上无“火星一号公路”字样。自驾游客在此停车进入遗址参观后出来，往往未仔细辨别方向，极易走上茶冷公路。
更具有迷惑性的则是，虽然是废弃多年的道路，误入茶冷公路后的一段距离，路况非常出色，反而是火星一号公路开始的路况看上去更像是“错路”。这就让很多走错的自驾游客走出去很远之后才发现自己走错了，而那时往往道路和补给都出了问题。
因此，我反复通过视频强调相关路面走错的问题，甚至拜托一位粉丝朋友亲自带着孩子前往该地自驾，并通过他拍摄的视频再次进行解读。
之所以这样做，是我觉得当我的流量大了之后，我需要进行一些这样的内容，网友通过观看我的视频，能避免事故的发生，这远比大家玩梗更有价值。这本不是我的义务，但现在我觉得我有责任这么做。
我从来不提倡没有准备就进入无人区穿越，但是我觉得这几期视频看完，至少人们不会走错路了。
要是看完还不知道怎么走的，这边建议多看几遍，不行可以下载下来，截图打印也行。
<strong>普通人“神秘园”</strong>我给自己的ID起名叫“神秘园”，主要还是喜欢那个欧洲乐队，这么说我可能暴露年龄了。
暴露了也没事，年龄可以大/大方方说，我现在30多岁，用互联网的话来说，算是个“中登”。
我并非网上传说中的户外“强驴”，勉强算是个户外运动爱好者，但我从来没有去尝试过冒险，更没爬过雪山，也从来没去过鳌太线。
业余时间最大的爱好是自驾游，但也没自驾过多远的地方，茶冷公路我都没去过。
去年5月的一天，我和朋友从河南郑州悄悄出发，经陕西、四川、贵州、广西、湖南、湖北最终返回郑州，跑了7个省转了一大圈。<img src="https://p.sda1.dev/30/1bec174296ca663108d0690488bb415e/640" id="aimg_Y8Z01" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)广西百色某洞穴图/受访者提供很放松，很舒适，很满足，很安全。
我全职做UP主的原因应该和很多人类似，创业失败了。
创业之前我在互联网公司工作，这在后来做视频的过程中也提供了一些帮助，因为我知道网友喜欢看什么，也知道他们因何而来，又因何而去。
最开始我做的视频几乎都是根据系列纪录片《空中浩劫》进行的解说。那时我的视频没有太多的特殊性，制作上也没什么门槛，这就造成了这类视频的可复制性很强，所以转型就成为必然的选择。
偶然间，看到了一个博主讲述洞穴探险类的故事，这是我感兴趣的方面，我发现他的播放量也不错，但有一个问题就是，我看完后觉得自己没看明白。
于是我开始分析他视频的问题，就发现最大的问题是他是露脸讲述故事，很像曾经电视上的一些讲故事的节目，这就造成了UP主的脸和其他画面切换的时候会显得非常突兀。
更重要的是，这造成了观看者注意力被打断，最终导致没看懂故事本身。
所以我开始思考，能不能做一个让所有人的注意力都聚焦在故事本身，过程中不被打断的故事产品。
这样我就开始了洞穴、潜水方面内容的探索，也就让我把空难题材放下了。而在进行洞穴、潜水方面题材视频的制作过程中，我又发现了一些问题，虽然相比起空难题材解说模式，提高了一些门槛，但受制于素材量的限制，依然会出现内容方面的同质化。
因为这些年户外登山、徒步、穿越等活动越来越被人们关注，我就将选题扩展到了更大的冒险方面，也就是现在大家看到的“神秘园”视频的样子。
虽然很多网友看“神秘园”后觉得自己“强得离谱”，但我做到现在反而胆子越来越小了，毕竟手上的“人命”太多了，自己肯定是增加了更多的敬畏感和理性。实话说，我之前做空难视频带来的“后遗症”还没有缓解，目前还不太敢坐飞机。
其实，户外运动是当下人们很好缓解工作压力的方式，大家接触户外，拥抱自然是非常健康的生活方式，但前提是，要明白“生活方式”四个字，前两个字一个是“生”，一个是“活”，咱得先活着才行。
我现在还没做好出镜露脸的准备，但一切皆有可能，保不齐哪天我就露脸了，给你们一个惊喜。
我衷心希望，大家平平安安的，能成为自己人生中的主角，而不是“神秘园”中的主角。
我郑重承诺，我“神秘园”一定不会在鳌太线的2800营地卖烤肠，半山腰没有卖烤肠的山大家也不要去爬。记者：胡克非编辑：胡韵运营编辑：马晓轶
