source: [https://gamesbeat.com/inside-a-s](https://gamesbeat.com/inside-a-s) ... ns-for-id-software/

id Software 团队原本似乎避开了 Xbox 裁员，因为该工作室此前并未收到可能被关闭或出售的通知。但就在周一，Xbox 宣布现有及计划裁员 3,200 人之际，这家微软旗下游戏工作室约一半员工遭到裁撤。

据我听到的消息，约 185 名全职员工中至少有 92 人被裁，超过员工总数的 50%。（社交媒体上流传的数字有所冲突，另外还有一些与 id 有关联的人员也遭到裁撤。）这让很多人感到意外，因为这家开创性的工作室拥有 Xbox 热门作品阵容中最重要的资产之一：重振后的《毁灭战士》系列。前员工表示，这场匆忙进行的 Xbox 裁员，其荒唐之处从被选中裁掉的人选上就能看出来。

“这就是一场血洗，”一名前员工说。“我完全被打了个措手不及。规模太大了。”

id Software 之所以显得脆弱，部分原因在于它正处于主要项目之间的空档期。就在前一天，该工作室刚刚发布了《毁灭战士：黑暗时代》的可下载内容，也就是资料片《Revelations》。裁员发生时，id 团队正尝试不同的游戏创意，包括一款代号为《Fury》的约翰·威克风格原创游戏；一款《完美黑暗》系列新作；以及一款多人/合作版《毁灭战士》。目前尚不清楚剩下的团队还能做哪一款，或者还能不能继续推进任何一个项目。

裁员带来的痛苦在整个 Xbox 网络中回荡。首席 UI 艺术家 Daniel Gagne 在 LinkedIn 发文称，他躲过了裁员。但他说：“现在要找到合适的词语真的非常困难。大量才华横溢、能力出众的人刚刚失去了自己的岗位。”

Gagne 补充道：“我在游戏行业走过了 25 年以上，也见过不少重组。但这一次刺痛得更深。看到朋友和同事——其中一些人为 id Software 奉献了二十多年——失去工作，令人心碎。3A 游戏生态必须改变；当前模式对于创造这些世界的人来说已经不可持续。”

你又回到了《毁灭战士：黑暗时代——Revelations》DLC 中的地狱。来源：id Software

这家位于得克萨斯州理查森、靠近达拉斯的工作室，并未像另外五家游戏工作室那样被列入关闭或出售名单。但据我从周一被裁员工那里听到的说法，如果 id 早知道裁员会如此严重，它可能会选择卖给其他买家。当天早上，工作室负责人 Marty Stratton 在 Teams 通话中向员工宣布裁员时哽咽了。那场通话只持续了几分钟，他表示这并不代表员工的技能或对工作室的影响力存在问题。

对于一家由四个年轻人于 1991 年在达拉斯创立的工作室来说，这个结果令人难过。这四个人是 John Carmack、John Romero、Adrian Carmack 和 Tom Hall。他们创造了《德军总部》《毁灭战士》和《雷神之锤》等游戏。后来他们各自走上不同道路，并在 2009 年将 id 出售给 ZeniMax。今年 8 月，他们将在达拉斯的 QuakeCon 上为《雷神之锤》30 周年举行重聚活动。

这些裁员显示出，Xbox 整体战略考虑不周，最终背弃了这家为长期粉丝制作单人游戏的工作室。正如前文提到的，Xbox Game Pass 原本预计现在应达到约 7,700 万订阅用户，但当前数字为 3,000 万，而且还在下降。

一名前工作室负责人表示，微软想要的目标令人困惑。到底是 Xbox Game Pass 订阅人数？Game Pass 上的高参与时长？单独销售游戏副本带来的收入和利润？还是平台技术演示？id Software 隶属于 Bethesda，而 Bethesda 的母公司 ZeniMax 于 2021 年被微软以 75 亿美元收购。

《毁灭战士：黑暗时代》在 Metacritic 上的综合评分为 83/100，并不差。问题在于，虽然有数百万玩家游玩了它，但许多人是通过 Game Pass 免费游玩的。《毁灭战士》有点像 Xbox 用来对标 PlayStation 游戏的作品，例如《战神》或《最后生还者》。但《毁灭战士：黑暗时代》的独立销售表现疲弱，而单独售卖本可以带来更多收入和利润。这让它成为裁员目标。如果 id Software 早知道目标是收入和利润，它也许会选择做服务型游戏或多人模式。

专注单人游戏是工作室为了维持较低员工规模而做出的选择。像 id 这样拥有数十年 3A 声誉的工作室规模如此之小，本身就很少见。这在沟通方面是一种优势。另一方面，id Software 往往需要四年时间制作一款游戏，比其他团队更久。也有人称，他们经常不得不加班赶工。游戏总监 Hugo Martin 常常告诉开发者，他希望他们成为真正的信徒，对制作伟大游戏怀有热情和投入。

id 还使用自己的游戏引擎 id Tech 8，这是 id 联合创始人 John Carmack 留下的遗产，不过 Carmack 已经离开。Carmack 当初创造 id Tech 游戏引擎，是为了让公司游戏在速度和视觉质量上拥有优势，而 id 至今仍在使用该引擎。它就像一辆为特定赛事打造的赛车，可以根据开发者需求定制，并能实现高帧率。然而，微软旗下游戏工作室，例如 Halo Studios，已经开始转向统一使用 Unreal Engine，这可能有助于降低成本并提升工程团队效率。

对 id Software 来说，这种转向可能存在风险，因为 id Tech 8 过去专注于大规模场景下的高速表现，而《毁灭战士》游戏需要这种速度。除此之外，熟悉 id Tech 8 的外包人员很可能并不多。

被裁员工承受了沉重打击。

“这真的让人非常紧张，我已经彻底精疲力尽了，因为我们刚刚完成 DLC。我们原本以为可能会受到影响，但没想到会到这种程度，”一名被裁员工说。

事实上，id Software 最新的可下载内容《Revelations》现在正随《毁灭战士：黑暗时代》推出。这是团队中一半人最后一次完成的软件作品。

受到严重冲击的部门包括 AI 编程、技术设计、其他设计部门、渲染、关卡设计、玩法设计、战斗设计、游戏引擎团队、环境美术、《雷神之锤：冠军》等。大量工会员工也被裁，而工会员工此前占工作室一半以上。我没有确切数字，但被裁人员中约 70% 是工会员工。

由于去年完成了大型项目《毁灭战士：黑暗时代》，团队中的外包人员并不多。远程员工也遭到裁撤，人数可能约为 35 人。公司正在与德国法兰克福团队沟通，以便遵守德国法律。幸存下来的部门包括过场动画、服务与开发运维、玩法工程。引擎团队可能只剩下几个人。似乎没有人在做《雷神之锤》。

至于代号为《Fury》的项目，那是游戏总监兼工作室联合总监 Hugo Martin 过去提出的原创 IP。它包含科幻、黑**、路易斯安那和芝加哥黑帮等元素，带有现代、类似赛博朋克的感觉。它还有一个叫“Gun Fu”的概念，把枪战与武术结合起来。这款游戏原本希望带来类似约翰·威克电影的体验，但并未正式获准进入制作阶段。

这是一次超越《毁灭战士》的机会，但远离核心系列显然并非微软当前想做的事。

团队也曾考虑制作一款基于《完美黑暗》的游戏。这个系列自 2005 年 Xbox 360 主机上的《Perfect Dark Zero》之后，就再没有新作。某天，有人在玻璃办公室里做了一场演示，工作室才听说这个想法。

位于圣莫尼卡的 Xbox 工作室 The Initiative 曾花了长达七年时间尝试制作一款《完美黑暗》游戏，后来 The Initiative 于 2025 年 7 月被关闭。显然，由于 id Software 可以使用其他微软旗下、尚未由别的团队开发的 IP，《完美黑暗》这个项目便有了可行性。相关概念艺术已经在制作中。

《毁灭战士：永恒》于 3 月 20 日登陆 PC 和主机。

微软以 75 亿美元收购了《毁灭战士：永恒》所有方 ZeniMax Media（Bethesda）。

另一个候选项目是一款西部风格机器人游戏，重点放在生存上，代号为《Ironwood》，其世界观类似《西部世界》。

至于《毁灭战士》，当时也有关于多人、合作以及更多 DLC 的提案。其中一个想法是让《毁灭战士：永恒》和 2016 年《毁灭战士》等旧作中的经典武器回归。幸运的是，id Software 仍有许多老员工能够做这些工作，因为它此前基本躲过了微软前几轮裁员。但现在也许不再如此。

在这些想法中，大多数都被视为面向本世代硬件的跨平台游戏。不过像《Fury》这样的项目可能开发周期过长，以至于最终需要瞄准微软即将推出的游戏主机 Project Helix。

根据美国联邦《WARN 法案》，员工在遭遇裁员时可获得 60 天通知期。但被裁的工会员工尚不清楚自己的离职补偿方案，因为相关内容还没有完成谈判。id 团队中的一些人正被调派到其他项目，例如 Arkane 的《Blade》或 MachineGames 的《德军总部 3》。Matt Booty 或 Asha Sharma 等高层领导似乎没有出现在 id 现场。

至于 AI，工作室内几乎没有证据显示它被使用，即使有，也主要用于较为琐碎、与游戏开发无关的任务。当前就业市场并不好，因此一名受访者表示，他预计会看到一些独立创业团队出现。

令人难过的是，公司最近才开始向在 id 工作满 10 年、15 年或 20 年的员工颁发水晶奖杯。id 仍有可能被出售，但目前没有迹象显示这正在发生。id 也有可能被用来制作其他 Bethesda 游戏，就像 Obsidian 已经成为《辐射》的支持工作室一样。

id Software 作为独立工作室的前景并不乐观——讽刺的是，除非它重新大量招人。

“我并不相信这里还有一条可行的前路，”一名被裁员工说。“感觉这就是一连串下意识的膝跳反应。”

一名被裁员工表示，他们为用规模不大的工作室和自有 id Tech 8 技术打造出 3A 游戏而感到自豪。

“对于我们做出的技术，以及我们发布的游戏，我没有任何遗憾。作为一个人数不多、却能推出 3A 大作的工作室，我们真的很厉害。我们做出了定义类型的作品，把一个实际上已经被认为死亡的系列重新启动。我为我们做过的事感到骄傲。我为这个工作室感到骄傲。我为仍然留在那里的人，也为已经离开的人感到骄傲。”
