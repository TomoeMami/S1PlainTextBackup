消息称索尼 PS5 Pro 将支持 PSSR 超分辨率技术，表现优于AMD FSR© 由 IT之家 提供

据称，所谓 PS5 Pro 的 GPU 将具有 300 TOPS 的机器学习算力（8bit 整数计算），以及 67 TFLOPS 的 16bit 浮点算力，FP32 单精度算力估计为 33.5 TFLOPS，是当前 PS5 的 3 倍以上。如此高的 GPU 算力表明其频率也会很高（2.18~2.45 GHz），之前的爆料显示 PS5 Pro 会采用定制 APU，其中 GPU 将启用 60 个 CU（总共 64 个）。在 MLID 放出的这份“Trinity 技术概述”文档中显示（“Trinity” 是 PS5 迭代机型的代号，自 2022 年初以来一直在开发中），支持机器增强版 TAAU (Temporal Anti Aliasing Upscale) 技术。这一技术类似于 DLSS 或 FSR，具有完整的 HDR 支持。该技术还将支持动态分辨率，并在未来更新支持最高 8K 分辨率输出。从 1080p 超分至 4K 分辨率需要 2ms 的 GPU 周期时间，这一方面仍需要进一步优化，而且这样的库只需要 250MB 的内存。与 AMD FSR 和 NVIDIA DLSS 一样，这一技术不需要针对每款游戏进行特意适配，这意味着开发人员只需通过官方软件开发工具包即可轻松地支持 PSSR。[url=]<img src="https://assets.msn.cn/staticsb/statics/latest/views/icons/ArticleImageFullscreen.svg" id="aimg_mJJnN" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)[/url]

索尼表示，PSSR 与 FSR2 相比可实现更高的画质，但目前消息来源只有这份文件，我们无法将其与其他超分辨率技术横向比较，但它应该不会弱于 FSR2。值得注意的是，AMD 的 FSR 不像 DLSS 那样可以使用机器学习算法来进行处理，所以优点是普适性更强，相比依赖于英伟达自家硬件的 DLSS 存在劣势。不过，AMD 之前确实暗示过在他们的超分技术中添加机器学习算法的可能性，只是目前还没有正式公布。[url=]<img src="https://assets.msn.cn/staticsb/statics/latest/views/icons/ArticleImageFullscreen.svg" id="aimg_JxoAl" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)[/url]
