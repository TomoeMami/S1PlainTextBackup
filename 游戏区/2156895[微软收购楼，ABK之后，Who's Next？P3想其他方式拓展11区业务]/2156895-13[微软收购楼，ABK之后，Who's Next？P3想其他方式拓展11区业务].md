Chris Dring:XBOX为什么需要一场残酷的重启

https://www.thegamebusiness.com/p/why-does-xbox-need-a-brutal-reset

每当一家公司收购另一家公司时，我的工作就是解释这如何契合其战略。任天堂收购Dynamo Pictures，是因为它想把IP拓展到游戏之外。索尼收购Bungie，是因为它想发展服务型游戏。Take-Two收购Zynga，是因为它想在移动端获得更强的存在感。

当然，你未必认同这些战略。但从公司的角度来看，这无非是设定一个目标，然后弄清楚需要什么来实现它。

众所周知，Game Pass从未真正走出硬核玩家群体，而且看起来也不太可能（至少短期内不会）。Xbox没能在PC上建立起一个大规模的平台，在移动端更是连试都没试过。与此同时，游戏串流在整个行业中仍然占比很小，尽管它正在增长。结果是，Xbox的利润率微乎其微。用新任首席执行官Asha Sharma的话说，“不能再这样下去了”。

Xbox曾在一项未能奏效的战略上投入了大量资金。它同时在太多战线上作战，而且有太多游戏属于相近品类。它需要聚焦。而聚焦不仅关乎你做什么，也关乎你不做什么。

关于Sharma是否做出了正确的选择，会有很多看法。有些人认为Xbox应该彻底转型为第三方发行商，但我不认为那能满足微软领导层的期望。

我的看法是，鉴于Xbox的优势所在，至少就目前而言，专注于主机是合理的。正如Matthew Ball上周对我们所说：“我们需要在PC上做得更好吗？是的。我们需要在移动端做得更好吗？是的。但在我们巩固好现有平台之前，我们不能要求发行商和玩家在我们落后、技术不足的其他平台上下注。”

我也认为投资于更大的IP系列是正确的决定。Xbox应该着眼于让《光环》和《战争机器》重新成为市场领先的IP系列，也应该再尝试一次《神鬼寓言》。它在《使命召唤》《魔兽世界》和《我的世界》方面确实面临挑战，应该从《辐射》和《上古卷轴》中挖掘更多价值。而像《盗贼之海》《禁闭求生》和《奥日》这样的IP，在游戏内外也都有机会。

这已经有很多要聚焦的了。那么，《燧石》《 Pentiment 》《南方午夜》或《天外世界》又该如何融入这一切呢？这类游戏有很多来自独立团队和第三方团队，Xbox有必要在内部开发它们吗？

这种聚焦，这种向主机的回归转向，将意味着削减。它将意味着关闭工作室（或者出售工作室——而在当前环境下，这恐怕会很困难）。对此人们感到愤怒。这是Xbox的失败。工作室只是按要求做事。而且，这是行业里许多人早就预见的失败。

但我不认为这是Xbox的终点。事实上，我认为这正是它一直在说的那个“回归”的一部分。只是这个回归需要的不仅仅是一个新标志和几款独占游戏。它还需要承认自己走错了路，调整方向，真正去耕耘它当下所处的市场，而不只是它预期未来会出现的那个市场。

而遗憾的是，这样做将会付出沉重的代价。

id Software 团队原本似乎避开了 Xbox 裁员，因为该工作室此前并未收到可能被关闭或出售的通知。但就在周一，Xbox 宣布现有及计划裁员 3,200 人之际，这家微软旗下游戏工作室约一半员工遭到裁撤。

据我听到的消息，约 185 名全职员工中至少有 92 人被裁，超过员工总数的 50%。（社交媒体上流传的数字有所冲突，另外还有一些与 id 有关联的人员也遭到裁撤。）这让很多人感到意外，因为这家开创性的工作室拥有 Xbox 热门作品阵容中最重要的资产之一：重振后的《毁灭战士》系列。前员工表示，这场匆忙进行的 Xbox 裁员，其荒唐之处从被选中裁掉的人选上就能看出来。

“这就是一场血洗，”一名前员工说。“我完全被打了个措手不及。规模太大了。”

id Software 之所以显得脆弱，部分原因在于它正处于主要项目之间的空档期。就在前一天，该工作室刚刚发布了《毁灭战士：黑暗时代》的可下载内容，也就是资料片《Revelations》。裁员发生时，id 团队正尝试不同的游戏创意，包括一款代号为《Fury》的约翰·威克风格原创游戏；一款《完美黑暗》系列新作；以及一款多人/合作版《毁灭战士》。目前尚不清楚剩下的团队还能做哪一款，或者还能不能继续推进任何一个项目。

裁员带来的痛苦在整个 Xbox 网络中回荡。首席 UI 艺术家 Daniel Gagne 在 LinkedIn 发文称，他躲过了裁员。但他说：“现在要找到合适的词语真的非常困难。大量才华横溢、能力出众的人刚刚失去了自己的岗位。”

Gagne 补充道：“我在游戏行业走过了 25 年以上，也见过不少重组。但这一次刺痛得更深。看到朋友和同事——其中一些人为 id Software 奉献了二十多年——失去工作，令人心碎。3A 游戏生态必须改变；当前模式对于创造这些世界的人来说已经不可持续。”

这家位于得克萨斯州理查森、靠近达拉斯的工作室，并未像另外五家游戏工作室那样被列入关闭或出售名单。但据我从周一被裁员工那里听到的说法，如果 id 早知道裁员会如此严重，它可能会选择卖给其他买家。当天早上，工作室负责人 Marty Stratton 在 Teams 通话中向员工宣布裁员时哽咽了。那场通话只持续了几分钟，他表示这并不代表员工的技能或对工作室的影响力存在问题。

对于一家由四个年轻人于 1991 年在达拉斯创立的工作室来说，这个结果令人难过。这四个人是 John Carmack、John Romero、Adrian Carmack 和 Tom Hall。他们创造了《德军总部》《毁灭战士》和《雷神之锤》等游戏。后来他们各自走上不同道路，并在 2009 年将 id 出售给 ZeniMax。今年 8 月，他们将在达拉斯的 QuakeCon 上为《雷神之锤》30 周年举行重聚活动。

这些裁员显示出，Xbox 整体战略考虑不周，最终背弃了这家为长期粉丝制作单人游戏的工作室。正如前文提到的，Xbox Game Pass 原本预计现在应达到约 7,700 万订阅用户，但当前数字为 3,000 万，而且还在下降。

一名前工作室负责人表示，微软想要的目标令人困惑。到底是 Xbox Game Pass 订阅人数？Game Pass 上的高参与时长？单独销售游戏副本带来的收入和利润？还是平台技术演示？id Software 隶属于 Bethesda，而 Bethesda 的母公司 ZeniMax 于 2021 年被微软以 75 亿美元收购。

《毁灭战士：黑暗时代》在 Metacritic 上的综合评分为 83/100，并不差。问题在于，虽然有数百万玩家游玩了它，但许多人是通过 Game Pass 免费游玩的。《毁灭战士》有点像 Xbox 用来对标 PlayStation 游戏的作品，例如《战神》或《最后生还者》。但《毁灭战士：黑暗时代》的独立销售表现疲弱，而单独售卖本可以带来更多收入和利润。这让它成为裁员目标。如果 id Software 早知道目标是收入和利润，它也许会选择做服务型游戏或多人模式。

专注单人游戏是工作室为了维持较低员工规模而做出的选择。像 id 这样拥有数十年 3A 声誉的工作室规模如此之小，本身就很少见。这在沟通方面是一种优势。另一方面，id Software 往往需要四年时间制作一款游戏，比其他团队更久。也有人称，他们经常不得不加班赶工。游戏总监 Hugo Martin 常常告诉开发者，他希望他们成为真正的信徒，对制作伟大游戏怀有热情和投入。

id 还使用自己的游戏引擎 id Tech 8，这是 id 联合创始人 John Carmack 留下的遗产，不过 Carmack 已经离开。Carmack 当初创造 id Tech 游戏引擎，是为了让公司游戏在速度和视觉质量上拥有优势，而 id 至今仍在使用该引擎。它就像一辆为特定赛事打造的赛车，可以根据开发者需求定制，并能实现高帧率。然而，微软旗下游戏工作室，例如 Halo Studios，已经开始转向统一使用 Unreal Engine，这可能有助于降低成本并提升工程团队效率。

对 id Software 来说，这种转向可能存在风险，因为 id Tech 8 过去专注于大规模场景下的高速表现，而《毁灭战士》游戏需要这种速度。除此之外，熟悉 id Tech 8 的外包人员很可能并不多。

被裁员工承受了沉重打击。

“这真的让人非常紧张，我已经彻底精疲力尽了，因为我们刚刚完成 DLC。我们原本以为可能会受到影响，但没想到会到这种程度，”一名被裁员工说。

事实上，id Software 最新的可下载内容《Revelations》现在正随《毁灭战士：黑暗时代》推出。这是团队中一半人最后一次完成的软件作品。

由于去年完成了大型项目《毁灭战士：黑暗时代》，团队中的外包人员并不多。远程员工也遭到裁撤，人数可能约为 35 人。公司正在与德国法兰克福团队沟通，以便遵守德国法律。幸存下来的部门包括过场动画、服务与开发运维、玩法工程。引擎团队可能只剩下几个人。似乎没有人在做《雷神之锤》。

这是一次超越《毁灭战士》的机会，但远离核心系列显然并非微软当前想做的事。

微软以 75 亿美元收购了《毁灭战士：永恒》所有方 ZeniMax Media（Bethesda）。

Jez的报道

 <blockquote><strong>Xbox 旗下的 id Software 现状如何？关于这家《毁灭战士》开发商破产的传闻被严重夸大了——以下是已知的事实。</strong>

不得不承认，Xbox 在过去一周里的裁员举措极为残酷。

已有四家工作室被剥离，第五家可能也危在旦夕；超过一千个岗位已被削减，据传未来还有2000个裁员计划。目前 Xbox 内部的氛围极其低落——这与今年春季新任 CEO 阿莎·夏尔马（Asha Sharma）上任时引发的乐观浪潮相比，简直是180度的大转弯。

因此，关于 Xbox 剩余工作室命运的各种传言四起。目前的很大一部分舆论都聚焦在了《毁灭战士（DOOM）》的著名开发商 id Software 身上。作为业界基石，该工作室无疑是现代游戏史上最重要、最具影响力的品牌之一。

id Software 此次遭遇的裁员不仅规模巨大，且令人痛心。有人声称，该工作室引以为傲的 id Tech 引擎实际上已被“雪藏”，且 id Software 目前的人员规模已不足以独立开发自己的游戏。

那么，真相究竟如何？我近期对微软针对 id Software 的规划进行了调查。尽管微软的财务人员手段冷酷，但我坚信这家工作室能够挺过难关。
<strong>微软无意搁置 id Tech 引擎</strong>

长久以来，《毁灭战士》及其他 id Software 的游戏都由 id Tech 引擎驱动。该引擎最初在90年代为《雷神之锤（Quake）》设计，此后为大量不同类型的游戏提供了支持，包括开放世界游戏《狂怒（RAGE）》、恐怖游戏《恶灵附身（The Evil Within）》、《德军总部（Wolfenstein）》系列射击游戏，当然还有现代的《毁灭战士》系列。最近，Machine Games 旗下的《夺宝奇兵》以及《毁灭战士：黑暗时代（DOOM: The Dark Ages）》都是基于最新版本的 id Tech 引擎构建的。

特别是在第一人称游戏中，它带给玩家的体验与虚幻引擎（Unreal Engine）有着显著的不同。该引擎因其在低端硬件配置上的神级优化以及极强的通用性而备受赞誉。

毫无疑问，对 id Software 的裁员严重冲击了负责开发 id Tech 的团队。但关于“该引擎目前几乎无人维护”的传闻是失实的。知情人士告诉我，在 id Software 本身以及 Machine Games 内部，依然保留着一支拥有 id Tech 核心技术实力的坚实团队。据透露，微软也没有强迫这两个工作室转向虚幻引擎的计划——而虚幻引擎目前似乎已成为微软及许多其他主流游戏开发商的默认选择。

微软针对该话题向我们发表了如下声明：

“在多个办公地点，仍有数十名员工在致力于 id Tech 引擎的开发工作。有关‘德克萨斯州总部只剩一名员工’的报道是不准确的。”

在我看来，微软如果停用 id Tech 将是极其短视的行为。将更多的主导权拱手让给虚幻引擎，可能会让他们在未来面临垄断性的涨价风险，这在财政上是不负责任的。Xbox CEO 阿莎·夏尔马本人在致员工的备忘录中也提到，她希望微软在类似技术上减少对外部供应商的依赖。因此，我认为微软应该加大对 id Tech 的投入，而不是减少。不过话说回来，眼光局限于季度财报正是微软如今的行事风格。
<strong>id Software 目前的人员规模与《毁灭战士（2016）》时期相当</strong>

另一个流传甚广的谣言是，id Software 的团队体量已经萎缩到无法支持未来独立开发产品的地步，从而被降级为“辅助型工作室”。这也是不实之词。

尽管这次裁员冷酷得令人心惊，但据我了解，id Software 目前留存的核心团队规模，与当年推出《毁灭战士（2016）》时大致相当。在我看来，尽管当时的开发团队规模相对较小，但《毁灭战士（2016）》堪称该工作室有史以来最出色的游戏。

不过，我目前还不确定 id Software 接下来会制作什么。有报道称，id Software 此前有一系列的提案正在筹备中，包括《完美黑暗（Perfect Dark）》的重启版，以及一款代号为“Ironwood”的西部风格射击游戏。目前尚不清楚这些项目是否会获得批准。

id Software 刚刚推出了《毁灭战士：黑暗时代 启示录（DOOM: The Dark Ages Revelations）》，这是去年那部《毁灭战士》前传的一款体量颇大的剧情扩张包。虽然工作室下一步的动作仍有待观察，但 Xbox CEO 阿莎·夏尔马曾明确点名《毁灭战士》和《雷神之锤》是值得持续投资的IP。

微软希望做大其最核心、最具辨识度的自有IP，这在短期内可能会将“全新”的IP创意排除在外。不过我还是很想看看现代版的《巫术神兵（Hexen）》会是什么样子……也许在照常举行的雷神之锤嘉年华（Quakecon）上，我们会了解到更多消息。
<strong>这依然是一场毫无必要的残酷阵痛</strong>

尽管关于 id Software 走向灭亡的传闻言过其实，但此次裁员依然显得冷酷而无情。即便现代《毁灭战士》三部曲取得了巨大的成功，许多拥有数十年丰富经验和资历的老员工还是被无情辞退。

Xbox 的游戏业务雄心在过去一周里遭遇了剧烈的航向修正，CEO 阿莎·夏尔马将其形容为一次“重置”。这导致了开发各种项目（从《上古卷轴》到 Xbox 的社交媒体分发及平台功能）的团队被大幅缩减。相比之下，暴雪等其他工作室目前似乎基本上毫发损未。此外，负责打造 Helix 的 Xbox 硬件团队也被完整保留了下来。

Xbox 曾努力保护 Undead Labs 等工作室免遭直接关闭的命运。但是，这并未能阻止微软在 Xbox 内部及其他领域抹杀数以百计的工作岗位。微软在人工智能（AI）领域的巨额赌注并未真正带来显著的收益，而对数据中心建设的庞大资本支出正让投资者感到恐慌。

在 Xbox 内部，裁员反映出传统游戏行业正面临着巨大的挑战。由于成本增加以及对年轻群体投资的匮乏，核心游戏市场出现了萎缩——年轻一代抛弃了主机，转而投向移动端上的《罗布乐思（Roblox）》以及其他非游戏活动。微软此前对 Xbox 平台及社交功能的投资不足，导致其无法在 Gen-X（X世代）和 Gen-Y（Y世代）之外拓展新用户，而这两个传统用户群体中的许多人已经开始步入不再频繁玩游戏的年龄段。

Xbox 接下来将走向何方仍完全是一个未知数，但有一件事是确定的：《毁灭战士》是永恒的（DOOM is eternal），它的生命力终将熬过微软所有的糟糕决策。</blockquote>
