Premium Ace Pass 常见问题解答

ACE COMBAT 8 提供的不同 ACE PASS 档位包含哪些内容？

ACE COMBAT 8 有三种 ACE PASS 类型。ACE PASS 是一条面向所有玩家开放的免费赛季成长通行证。玩家在升级时即可获得奖励，包括外观道具和机体升级。

还有 PREMIUM ACE PASS，它在每个等级都会提供额外奖励，并包含诸如机体 ADFX-02 Morgan–PIXY 等高级赛季内容，该机体为第 1 赛季独占。

此外还有一个更高档位，称为 PREMIUM ACE PASS PLUS，其中包含 PREMIUM ACE PASS 的奖励，以及 30 级跳过。玩家可以立即解锁 30 级的赛季成长与奖励。

关于 ACE PASS 各档位所包含的季节奖励细节以及即将推出的内容，将在稍后日期公布。

可通过《ACE COMBAT 8》豪华版中包含的兑换券解锁高级 ACE PASS PLUS。两个高级 ACE PASS 档位也都可单独购买。

豪华版附带的兑换券有什么作用？

豪华版包含一张可供玩家使用的代金券，可兑换一个赛季的“高级 ACE PASS PLUS”内容和奖励。该代金券不绑定特定赛季，这意味着玩家可以将其保存起来，并在愿意时用于兑换未来的“高级 ACE PASS PLUS”赛季内容。

高级 ACE PASS 解锁内容是否包含能提升游戏性的道具？

我们提供一项特别奖励，用于庆祝《ACE COMBAT 8》以及第 1 季“高级 ACE PASS”的上线。订购豪华版并使用代金券兑换“高级 ACE PASS”的玩家，将立即解锁 ADFX-02 Morgan–PIXY，该机型也可在在线模式中游玩。若玩家希望在战役模式中使用该机体，需要完成游戏的战役模式。

此外，该通行证还可让你访问各种令人兴奋的外观道具，包括机体涂装以及游戏内头像服装。

《ACE COMBAT 8》会包含DLC吗？

ACE COMBAT 系列长期以来一直通过在发售后提供额外内容来支持玩家。虽然我们目前还准备好分享未来内容计划的细节，但我们期待在未来分享更多信息。

《ACE COMBAT 8》可购买的物品是否包含消耗品？

目前没有计划将消耗品加入游戏，无论是作为购买内容，还是作为 ACE PASS 奖励的一部分。将提供的奖励与内容为玩家提供购买或解锁可玩飞机的方式、用于升级飞机的道具，以及用于外观装饰的物品。

是否有基于概率的购买和/或类似“战利品箱”的机制？

《ACE COMBAT 8》不包含付费战利品箱机制，或其他随机的付费奖励。玩家可以在购买之前查看所有可用内容和奖励。

《ACE COMBAT 8》中的高级Ace通行证（Premium Ace Pass）与《ACE COMBAT》以往的战斗通行证有什么不同？

《ACE COMBAT 7》的赛季通行证提供可游玩的战机以及额外的特殊任务。

《ACE COMBAT 8》的高级 ACE PASS 提供更多样的内容，包括可游玩的战机、外观物品，以及在通过通行证进度时作为奖励用于升级战机的物品。

《豪华版》是否仅在预购期间可用，还是我在发售后也能购买？

《豪华版》将在《ACE COMBAT 8》发售后仍可供购买。

请注意，不过它不包含《豪华版》数字预购所提供的提前体验奖励。
