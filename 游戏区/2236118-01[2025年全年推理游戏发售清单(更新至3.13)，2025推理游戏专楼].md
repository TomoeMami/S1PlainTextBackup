笑死，刚过完断刀，在专楼来找其他的悬疑类文字ADV，就看到有楼友评价断刀的内容和我在群里吐槽的几乎一样，都是一开始就想到七凭之祭然后一直等师姐爆个大的。果然悬疑类作品看多以后再遇到类似点子新鲜感和趣味性就会少很多

我对这游戏最大的意见是总喜欢玩网络烂梗，时不时还加点meta吐槽，真的无趣/尬而且出戏，这对整个题材还算严肃的作品代入感的破坏可以说是毁灭性的。后续故事能把前面的穿起来一口气看完还是很流畅的，但前面的漏洞给的实在有些明显加上结束的很仓促，以至于部分设定没用上我还以为是自己想复杂了。用一个架空高武的武侠世界背景给作案手法填坑的做法也不算太高明。还总有个弹丸十神定位的逼王角色给玩家上血压，甚至更过分。人家是暴风雪山庄连环谋杀案，他搁这儿度假来了，对主线的发展也没看出有什么影响，甚至直接随便捏死蚂蚁一样在部分结局干死玩家，真的膈应人。不管怎么说，得益于后半部分的节奏很紧凑，体验总的来说还行，接下来看看三相奇谈

2025.7.14

神秘推理新游《Project Code M》揭开面纱，推理游戏月报2025.7上

各位好啊，这边是还在越南喂小强的火轮，前段时间我刚把《百日战纪》的所有100个结局打通了，评价为小高和刚 好大喜功又能力不足，最后做成项目灾难...来看本月推理游戏新闻吧。
1.《海市蜃楼之馆》开发组NOVECT，在Anime Expo 2025公布推理题材新作《Project Code M(暂名)》预告片，本作包含清理员与杀人犯双视角叙事，开发周期长达七年，计划登陆多平台，发售日期待定。
这游戏早在2022年就已经公布，一转眼就是三年了，这游戏终于有动静了，虽然发售日期还是写着TBD...
2.在Anime Expo 2025上，Aksys Games宣布代理推理游戏《ミステリーの歩き方》英文版，起名为《Path of Mystery: A Brush with Death(推理谜径:画师之逝)》，计划2026春发布NS版，与此同时《Project Code M》也确认由该会社代理英文版。
Aksys代理过不少推理游戏，最经典的就是“极限脱出三部曲”，金沢十三男之前的乙女悬疑游戏《七罪绯红》也是该社代理，所以本作的代理名选也不意外，不过Toybox真的不打算出个中文？
3.密室逃脱游戏《残音系少年少女》宣布将于9月17日发售，支持中文。本作为青春伤痕题材解谜游戏，6名少年少女在记忆的迷宫里寻找最终的真相。
有公开试玩，游戏美术不错，不过游戏剧情感觉看到开头就能猜到结尾，谜题设计也有点无趣，也许是我太老登了，这种题材已经离我太远了...
4.在《壳之少女》17周年之际，Innocent Grey社长杉菜水姬在推特吐露，目前正在规划推出“时坂侦探事务所”背景的新故事。“时坂侦探事务所”为美少女悬疑游戏“壳之少女”系列三部曲的场景，此消息疑似为系列新作预告。
IG社这次终于不卖清水百合了，泪目，但这个描述感觉就很模糊，不一定是游戏新作，谨慎看待吧。
5.《奥术律师》开发者Diamondhenge宣布，将对本作进行重制计划，持有原版游戏的玩家可得到免费升级，重制后本作将提价，计划几个月后推出。
我以为重制的日期会很靠后，现在看这部分工作日期还挺近的...刚出没几个月的汉化补丁看着要报废了。
6.由@萨格罗斯独立开发，数据库检索推理游戏《述罪》现已在steam发布正式试玩版，本作玩法类似《真探》，通过关键字检索系统来推进故事发展。
受朋友之托来介绍一下，本人有幸参与了本作上半年的内测，相比《真探》而言，本作的黑白系画风更加令人印象深刻，玩法上也较为复杂一些，对这类数据库检索推理玩法的人可以在steam上尝试demo。
7.本月众筹情况
横版悬疑游戏《三相奇谈》周边已于摩点众筹RMB 157.5w，计划8月发货。
有钱到可疑的兽人控.jpg，但这确实是这个游戏应得的。
8.本月推荐游戏——《鲤氏奇人异闻录》
明日方舟的免费同人作品，虽然我没接触过原作不过也不影响剧情理解，作为同人来说制作还算精良，诡计比较简单但故事还不错，可以试试。
